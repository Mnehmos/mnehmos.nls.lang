## Quick Start Checklist

Before starting the tutorials, make sure you have:

- [ ] Python 3.11+ installed
- [ ] `nlsc` installed (`pip install nlsc`)
- [ ] A code editor (VS Code recommended with NLS extension)

## Tutorial Format

Each tutorial includes:

1. **Goal** — What you'll build
2. **Concepts** — New ideas introduced
3. **Step-by-step** — Guided implementation
4. **Exercises** — Practice challenges
5. **Next steps** — Where to go from here
